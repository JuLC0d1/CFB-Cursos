Um pouco sobre: a

TARGET

  1. _blanck = abrir em outra pagina
  2. _parent = relacionado a conjunto de frameset
  3. _self = é o padrão, abre na mesma janela, mas não é obrigatorio usalo
  4. _top = relacionado a conjunto de frameset
  5. framename = se eu quiser especificar o nome me um quadro que está no meu conjunto(para usalo é só colocar o nome que deseja, por exemplo: caixa)

REL

Tipo de arquivo que vai ser chamado pelo link.

  1. alternate
      2. Link de representação alternativa de documento
  2. Author
      3. Oferece um link para uma pagina com informações do author
  3. bookmark
      4. URL permanente usado para bookmark = favorito
  4. external
      5. indica que o link mensionado não faz parte do documento atual

MEDIA

Para que tipo de midia este arquivo vai ser aberto, qual é mais adequado

  1. all(padrão)
      2. adequado para todos os dipos de midia/dispositivos
  2. aural
      3. sintetizadores de fala auditiva
  3. braille
      4. dispositivos de braille
  4. handheld
      5. dispositivos de tela pequena e banda limitada
  5. projection
      6. projetores
  6. print
      7. impressoras
  7. screen
      8. telas de computadores
  8. tv
      9. televisores
